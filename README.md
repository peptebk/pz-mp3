# pz-mp3

Приложение для ухудшения качества аудио. Конвертирует аудиофайлы в формат с низким битрейтом и частотой дискретизации.

## Функциональность

- Выбор входного аудиофайла (MP3 или WAV)
- Конвертация в MP3 с параметрами:
  - Частота дискретизации: 800 Гц
  - Количество каналов: 1 (моно)
  - Битрейт: 8k
  - Кодек: libmp3lame
- Сохранение результата в указанный файл

## Требования

- Python 3.x
- Библиотеки: tkinter, customtkinter
- FFmpeg (должен быть установлен в системе и доступен в PATH)
